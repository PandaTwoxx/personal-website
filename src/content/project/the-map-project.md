# Map

Map platform that stores friend locations

# Setup guidelines

1. Clone the repository with
```bash
```
2. Run the following commands:

```bash
poetry install
poetry shell
python3 manage.py run
```

3. Or run

```bash
docker compose up --build
```

4. Open the browser and go to `http://localhost:8080/`
