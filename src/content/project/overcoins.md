# Overcoins

Overcoins is a cool polling site where users can create and participate in various polls. You can earn overcoin currency by participating in polls. You can also create polls and load them with overcoins in order to incentivize participation. I have also left implementation available for future abilities to add or use a crypto wallet.

Try it out today [Overcoins](https://overcoins.tickitechnology.com)

Side note, if you have an existing overcast account, you can login with it.
